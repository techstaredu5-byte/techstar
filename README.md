# TechStar Portal Starter

Starter website + school management portal structure for TechStar Institute of Business Technology.

## Quick start

1. Import `database/techstar.sql` into MySQL.
2. Update database credentials in `config.php`.
3. Serve this folder with PHP (e.g. `php -S localhost:8000`).
4. Open `/index.php` and login with `admin` / `admin123`.
